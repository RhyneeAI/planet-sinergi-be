# v1.7 — Operational: Employee Management, Lembur, Kasbon, & Edit Log Jabatan

## Employee Management

### List Karyawan

`GET /api/v1/operational/employees` — response sama seperti `abs/employees` (semua user)

| Field tambahan | Notes |
|---|---|
| is_active | Status aktif/nonaktif |
| sub_company | Sub-company jika punya |

### Detail Karyawan

`GET /api/v1/operational/employees/{uuid}` — profil + riwayat kasbon + lembur + gaji

### Toggle Active

`PUT /api/v1/operational/employees/{uuid}/toggle-active`

Request: (no body)
Response: `{ "is_active": true/false }`

### Create Karyawan

`POST /api/v1/operational/employees` — Admin & HRD

Body:
```json
{
  "name": "...",
  "email": "...",
  "phone": "...",
  "password": "...",
  "role": "ADMIN|HRD|KASIR|MANAJER_GUDANG|MARKETING_LEAD|MARKETING|MARKETING_TETAP|MANDOR",
  "sub_company_uuid": null,
  "abs_jabatan_id": null,
  "abs_shift_id": null
}
```

> **Tidak bisa** create Superadmin atau Owner dari endpoint ini.

## Lihat Lembur (Read-only)

`GET /api/v1/operational/overtimes` — semua lembur approved
Filter: date range, user

## Lihat Kasbon (Read-only)

`GET /api/v1/operational/loans` — semua kasbon
Filter: status, user

## CRUD Jabatan (endpoint tetap di prefix Absensi)

**Route:** `/api/v1/abs/jabatans` (sudah ada, tambah edit log)

**Edit Log:** Gunakan table `ops_edit_logs` (polymorphic)

Trigger: saat field `salary` berubah:
```php
OpsEditLog::create([
    'loggable_type' => 'abs_jabatans',
    'loggable_id' => $jabatan->id,
    'reason' => $request->reason ?? '-',
    'old_data' => ['salary' => $old],
    'new_data' => ['salary' => $new],
    'edited_by' => auth()->id(),
    'company_id' => auth()->user()->company_id,
]);
```

### Middleware / Route Handling

Buat middleware `CheckModule`:

```php
class CheckModule
{
    public function handle($request, $next, $module)
    {
        if (!config("modules.{$module}.enabled", true)) {
            return response()->json([
                'success' => false,
                'message' => "Module {$module} tidak tersedia",
            ], 410);
        }
        return $next($request);
    }
}
```

Registrasi di `bootstrap/app.php`:

```php
->withMiddleware(function (Middleware $middleware) {
    $middleware->alias([
        'module' => \App\Http\Middleware\CheckModule::class,
    ]);
})
```

Gunakan di route group:
```php
Route::middleware('module:pos_customer')->group(function () {
    // customer routes
});
```
