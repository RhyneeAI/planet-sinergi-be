## Product ‚Äî New Pricing Fields

### Field Migration

| Field Lama | Field Baru | Keterangan |
|---|---|---|
| `purchase_price` | **`base_price`** | Modal perusahaan |
| `wholesale_price` | _hapus_ | |
| `retail_price` | _hapus_ | |
| `reseller_price` | _hapus_ | |
| ‚Äî | **`leader_price`** | Harga jual perusahaan ke LEAD_MARKETING / MARKETING_TETAP |

### Profit Flow

**Contoh: base=20k, leader=25k, marketing=26k, sell=30k**

| Skenario | Company | LEAD_MARKETING | MARKETING |
|---|---|---|---|
| Dijual oleh **MARKETING** | 5k (25-20) | 1k (26-25) | 4k (30-26) |
| Dijual oleh **MARKETING_LEAD** | 5k (25-20) | 5k (30-25) | ‚Äî |
| Dijual oleh **MARKETING_TETAP** | 5k (25-20) | ‚Äî | ‚Äî (bonus terpisah) |
| Dijual oleh **KASIR** | 5k (25-20) | ‚Äî | ‚Äî |

## Sales Transaction v2

### Changes

| Item | Sebelum | Sesudah |
|---|---|---|
| Customer | required | **nullable** (tidak perlu pilih) |
| Marketing | ‚Äî | **required** (`marketing_id`: LEAD_MARKETING / MARKETING / MARKETING_TETAP) |
| Discount | optional | tetap ada |
| Biaya opsional | optional | tetap ada |
| Custom sell_price | optional | tetap ada |

### Profit Calculation

Profit dihitung saat transaksi:
```php
$companyProfit = ($leaderPrice - $basePrice) * $qty;
$leadProfit = ($marketingPrice - $leaderPrice) * $qty; // jika MARKETING
$marketingProfit = ($sellPrice - $marketingPrice) * $qty;
```

Simpan di `pos_sales_details` sebagai field baru: `company_profit`, `lead_profit`, `marketing_profit`
