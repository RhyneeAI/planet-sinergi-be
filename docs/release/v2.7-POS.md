# v2.7 — POS: Role Restrukturasi (KEPALA_GUDANG, GUDANG, KEPALA_MANDOR)

## Ringkasan

Restrukturasi role POS:
- **MANAGER_GUDANG** → **GUDANG** (stock keeper)
- **KEPALA_GUDANG** — role baru, all access POS
- **KEPALA_MANDOR** — full POS access (dari v2.6)
- Marketing CRUD pindah ke OPS, POS read-only
- 2 gateway marketing: `/pos/marketings` (read) + `/ops/marketings` (CRUD)

## Marketing — 2 Gateway

| Endpoint | Module | Method | Keterangan |
|----------|--------|--------|------------|
| `GET /pos/v1/marketings` | POS | index | Read-only |
| `GET /pos/v1/marketings/{uuid}` | POS | show | Read-only |
| `GET /ops/v1/marketings` | OPS | index | List marketing |
| `POST /ops/v1/marketings` | OPS | store | Create marketing |
| `GET /ops/v1/marketings/{uuid}` | OPS | show | Detail marketing |
| `PATCH /ops/v1/marketings/{uuid}` | OPS | update | Edit marketing |
| `DELETE /ops/v1/marketings/{uuid}` | OPS | destroy | Hapus marketing |

## Yang Berubah

### Role Enum (`app/Enums/Role.php`)
- `MANAGER_GUDANG` → `GUDANG` (rename case & value)
- Tambah `KEPALA_GUDANG`
- Update label(), values()

### Factory
- Update `UserFactory.php` — rename state `managerGudang` → `gudang`, tambah `kepalaGudang`

### Routes POS (`routes/pos-api.php`)
- Semua `MANAGER_GUDANG` → `GUDANG,KEPALA_GUDANG,KEPALA_MANDOR`
- Grup read: `SUPERADMIN,OWNER,ADMIN,KEPALA_GUDANG,KEPALA_MANDOR,GUDANG`
- Grup write master data: kurangi jadi `SUPERADMIN,ADMIN,KEPALA_GUDANG,KEPALA_MANDOR` (GUDANG tidak bisa write supplier, customer, customer-type)
- Atau: GUDANG hanya bisa write categories, units, products (stock)
- Grup transaksi: `SUPERADMIN,OWNER,ADMIN,KEPALA_GUDANG,KEPALA_MANDOR,KASIR`
- Marketing endpoints: index/show → `SUPERADMIN,OWNER,ADMIN,KEPALA_GUDANG,KEPALA_MANDOR,GUDANG,KASIR,MARKETING_LEAD,MARKETING`; write dihapus

### Routes OPS (`routes/operational-api.php`)
- Tambah grup `marketings` resource → hanya CRUD (no index/show? atau full read-write?)
- Middleware: `SUPERADMIN,ADMIN,KEPALA_MANDOR` atau `SUPERADMIN,ADMIN`

### OPS Marketing Controller (baru)
